# Showcase Checklist

Use this checklist before publishing or sharing your repository.

- Add a short demo GIF (30-60 seconds) of chat mode and API requests.
- Pin this project on your GitHub profile.
- Add project link to resume and LinkedIn.
- Include 3-5 concrete outcomes in README (architecture, testing, automation).
- Keep tests green and CI status passing.
- Add tagged releases (`v1.0.0`, `v1.1.0`) with release notes.
- Add future milestones in issues and project board.
